Passo a Passo Completo

Siga este guia detalhado para criar seu primeiro projeto React com Vite

01

Criar o Projeto

Use o comando npm create vite para iniciar um novo projeto React. Este comando usa o template oficial do Vite.

Terminal 
# Criar projeto com template React
npm create vite@latest my-react-app -- --template react

# Ou de forma interativa:
npm create vite@latest
Dica: Você pode escolher entre react (JavaScript) ou react-ts (TypeScript).
02

Navegar até o Diretório

Entre na pasta do projeto recém-criado.

Terminal 
cd my-react-app
03

Instalar Dependências

Instale todos os pacotes necessários especificados no package.json.

Terminal 
# Com npm:
npm install

# Ou com yarn:
yarn

# Ou com pnpm:
pnpm install
Baixando pacotes
Instalando
Concluído
04

Iniciar o Servidor de Desenvolvimento

Execute o servidor de desenvolvimento para visualizar sua aplicação React em tempo real.

Terminal 
npm run dev
Servidor iniciado! Acesse http://localhost:5173 no seu navegador.
05

Estrutura do Projeto

Entenda a estrutura de arquivos gerada pelo Vite:

my-react-app/
node_modules/
public/

App.jsx
App.css
main.jsx

package.json
Arquivos principais:
  • main.jsx - Ponto de entrada da aplicação
  • App.jsx - Componente principal
  • vite.config.js - Configurações do Vite
06

Editar o Componente App

Vamos criar um componente simples para testar. Abra src/App.jsx e substitua pelo código:

src/App.jsx 
import { useState } from 'react'
import './App.css'

function App() {
  const [count, setCount] = useState(0)

  return (
    <div className="App">
      <h1>Olá, React + Vite! 🚀</h1>
      <div className="card">
        <button onClick={() => setCount((count) => count + 1)}>
          Contador: {count}
        </button>
      </div>
      <p className="read-the-docs">
        Edite src/App.jsx e salve para testar HMR
      </p>
    </div>
  )
}

export default App
07

Build para Produção

Quando estiver pronto para publicar, crie uma versão otimizada para produção:

Terminal 
npm run build
Os arquivos otimizados serão gerados na pasta dist/. Para testar localmente:
Terminal 
npm run preview

Perguntas Frequentes

Por que Vite é mais rápido que Create React App?

Vite usa ESM (ES Modules) nativos do navegador durante o desenvolvimento, eliminando a necessidade de bundling. Isso resulta em inicialização instantânea e HMR extremamente rápido, independentemente do tamanho do projeto.

Posso usar TypeScript com Vite?

Sim! Use o template react-ts ao criar o projeto: npm create vite@latest my-app -- --template react-ts

Como adicionar bibliotecas como React Router ou Zustand?

Use npm para instalar qualquer biblioteca: npm install react-router-dom ou npm install zustand

O Vite funciona com outras frameworks além de React?

Sim! Vite suporta Vue, Svelte, Preact, Lit e vanilla JavaScript. É uma ferramenta de build agnóstica a frameworks.

Como configurar variáveis de ambiente?

Crie um arquivo .env na raiz do projeto e use variáveis prefixadas com VITE_. Exemplo: VITE_API_URL=https://api.example.com

Onde posso fazer deploy do meu projeto?

Você pode fazer deploy em Vercel, Netlify, GitHub Pages, Render ou qualquer serviço de hospedagem estática. Basta executar npm run build e fazer upload da pasta dist/.
